# Power BI Package

No `.pbix` is included for this project yet. Power BI Desktop is installed, but creating and saving a new `.pbix` requires a manual GUI build; I did not create a placeholder file.

Included:

| File/folder | Purpose |
|---|---|
| `../data/powerbi/` | CSV inputs generated by `python scripts/run_sql.py`. |
| `dashboard_brief.md` | Dashboard purpose and pages. |
| `data_model.md` | Tables and relationships. |
| `dax_measures.md` | Measures to create. |
| `manual_build_instructions.md` | Rebuild guide. |
| `refresh_instructions.md` | Refresh workflow. |
| `screenshots/page*_mockup.png` | Static mockups from included data, not Desktop screenshots. |

The dashboard should focus on the risk-strategy tradeoff: fraud capture, review volume, false positives, and expected cost.
